# Agrotech Management System

A desktop-based management system developed using Core Java and JDBC for handling inventory, reporting, and agricultural business operations efficiently.

---

## Features

- Inventory Management
- Product Record Management
- Database Connectivity using JDBC
- Report Generation
- User-Friendly Desktop Interface
- CRUD Operations
- Optimized SQL Query Handling

---

## Tech Stack

### Backend
- Core Java
- JDBC

### UI
- Swing
- AWT

### Database
- MySQL

### Tools
- Eclipse
- Git & GitHub

## Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/yourusername/agrotech-management-system.git
```

2. Open project in Eclipse or IntelliJ IDEA

3. Configure MySQL database connection

4. Import required JDBC driver

5. Run the main Java file
